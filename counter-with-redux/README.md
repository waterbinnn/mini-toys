### install

```
npm install redux
npm install react-redux

// createStore 에 취소선 생기는 문제 - 기존 리덕스 제거 후 버전 맞춰줌
npm remove redux react-redux
npm install redux@4.1.2 react-redux

npm install --save styled-components
npm install styled-reset
```
